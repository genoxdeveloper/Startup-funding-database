## 🏢 เกี่ยวกับ Genox & Partners

**Genox**เป็นบริษัทร่วมทุนด้านนวัตกรรมด้านเทคโนโลยีที่ตั้งอยู่ในกรุงโซล ประเทศเกาหลีใต้ เราออกแบบโซลูชันและแพลตฟอร์มที่ขับเคลื่อนด้วยข้อมูลซึ่งเสริมศักยภาพให้กับสตาร์ทอัพระดับโลก เราเชื่อในการเข้าถึงโอกาสที่เป็นประชาธิปไตย ทลายขอบเขต และเร่งสร้างนวัตกรรม

โครงการโอเพ่นซอร์สนี้ได้รับการสนับสนุนอย่างภาคภูมิใจและพัฒนาร่วมกันโดยความร่วมมือกับ**[Buygit.com](https://buygit.com)**ซึ่งเป็นแพลตฟอร์มชั้นนำที่อุทิศตนเพื่อเสริมศักยภาพให้กับนักพัฒนาและผู้สร้างทั่วโลก Genox และ Buygit.com ร่วมกันมุ่งมั่นที่จะมอบเครื่องมือและข้อมูลที่จำเป็นแก่ผู้ก่อตั้งเพื่อสร้างสตาร์ทอัพระดับยูนิคอร์นรุ่นต่อไป

เราใช้เวลานับไม่ถ้วนในการค้นหาพอร์ทัลของรัฐบาล ฐานข้อมูล VC และไซต์เร่งความเร็วในหลายสิบประเทศ เราสร้างเครื่องมือนี้เป็นการภายในก่อน และตอนนี้เรากำลังเปิดซอร์สเพราะ**ผู้ก่อตั้งทุกคนสมควรได้รับการเข้าถึงโอกาสระดับโลก**ไม่ใช่แค่ผู้ก่อตั้งใน Silicon Valley

## 🚀 ไซต์นี้ให้คุณทำอะไรได้บ้าง?

การค้นหาเงินทุนหรือโปรแกรมการสนับสนุนที่เหมาะสมอาจมีเรื่องล้นหลาม โดยเฉพาะอย่างยิ่งเมื่อมองข้ามพรมแดน แอปพลิเคชั่นนี้แก้ปัญหานั้นด้วยการยกของหนักให้กับคุณ:

### 1. ค้นพบ Global Funding ทันที
สำรวจฐานข้อมูลขนาดใหญ่ที่อัปเดตอย่างต่อเนื่องของ:
-**เงินอุดหนุนจากรัฐบาล**(เช่น SBIR ในสหรัฐอเมริกา Innovate UK, K-Startup ในเกาหลี, Horizon Europe)
-**VCs & Accelerators**(Y Combinator, Techstars, 500 Global และ micro-VC ระดับภูมิภาคนับพัน)
-**โปรแกรม Corporate Open Innovation (OI)**
-**เครดิตและสิทธิพิเศษบนคลาวด์**(AWS Activate, Google สำหรับสตาร์ทอัพ)
- โครงการริเริ่ม**การย้ายถิ่นฐานและการเติบโต**(วีซ่าสตาร์ทอัพ, ที่อยู่อาศัยของ Tech Hub)

### 2. รองรับหลายภาษา (ทำลายขอบเขต)
เราตระหนักดีว่ายูนิคอร์นตัวใหญ่ตัวต่อไปอาจมาจากที่ใดก็ได้ อย่างไรก็ตาม ในอดีตระบบนิเวศของสตาร์ทอัพทั่วโลกถูกกีดขวางด้วยอุปสรรคทางภาษาที่รุนแรง พอร์ทัลการระดมทุนที่สำคัญ รายละเอียดการปฏิบัติตามข้อกำหนดการให้ทุนที่ซับซ้อน และวิทยานิพนธ์การลงทุน VC ที่ซับซ้อน มักจะถูกฝังไว้ทั้งหมดเป็นภาษาอังกฤษหรือภาษาถิ่น สิ่งนี้สร้างความได้เปรียบที่ไม่ยุติธรรมโดยอิงตามภูมิศาสตร์และภาษาแม่เพียงอย่างเดียว

เพื่อให้มั่นใจว่าจะไม่มีผู้ก่อตั้งคนไหนถูกทิ้งไว้ข้างหลัง แพลตฟอร์มของเราจึงมี**ระบบสนับสนุนหลายภาษาแบบเนทีฟ**ที่ทรงพลังอย่างเหลือเชื่อและเป็นผู้นำในอุตสาหกรรม นี่ไม่ใช่แค่การแปล UI ธรรมดาเท่านั้น แต่ยังเป็นไปป์ไลน์การแปลเชิงลึกที่ครอบคลุมอีกด้วย

ด้วยการคลิกเพียงครั้งเดียวที่แถบนำทางด้านบนของเรา คุณสามารถแปลทั้งแพลตฟอร์มได้อย่างราบรื่น ซึ่งหมายความว่า**คำอธิบายโปรแกรมที่ฝังลึกมากกว่า 35,000 รายการ เกณฑ์การประเมิน จุดเน้นของอุตสาหกรรม และรายการฐานข้อมูลแบบไดนามิก**ทั้งหมดจะถูกแปลเป็นภาษาท้องถิ่นทันที:

**ทำไมถึงปฏิวัติครั้งนี้?**
ฐานข้อมูลเริ่มต้นส่วนใหญ่จะแปลเฉพาะ UI แบบคงที่ (ปุ่ม เมนู) ในขณะที่ปล่อยให้ข้อมูลจริง (ซึ่งสำคัญที่สุด) เป็นภาษาต้นฉบับ แพลตฟอร์มของเราแก้ไขปัญหานี้โดยใช้**สถาปัตยกรรมการแปลแบบสองกลไก**:
1.**การแปล UI แบบคงที่:**ขับเคลื่อนโดย `Flask-Babel` เพื่อให้มั่นใจว่าองค์ประกอบโครงสร้าง การนำทาง และ UX หลักทั้งหมดได้รับการแปลอย่างสมบูรณ์แบบโดยมีเวลาแฝงเป็นศูนย์โดยใช้แค็ตตาล็อกข้อความ `.po` และ `.mo` ที่คอมไพล์ไว้ล่วงหน้า
2.**การแปลข้อมูลแบบไดนามิก:**ขับเคลื่อนโดยไปป์ไลน์อะซิงโครนัสที่ใช้ `ตัวแปลเชิงลึก' เมื่อคุณเปลี่ยนภาษา ระบบของเราจะดึงคำอธิบายโปรแกรมแบบสดและไม่มีโครงสร้างจากฐานข้อมูล SQLite ของเรา และแปลได้ทันที โดยรักษาสคีมาข้อมูลที่แน่นอน การจัดรูปแบบมาร์กดาวน์ และเกณฑ์การจัดหาเงินทุนที่สำคัญ

ซึ่งหมายความว่าผู้ก่อตั้งในโบโกตา ริยาด หรือโซลสามารถอ่านข้อกำหนดการปฏิบัติตามข้อกำหนดที่ซับซ้อนของทุน SBIR ของสหรัฐอเมริกา หรือวิทยานิพนธ์การลงทุนของเครือข่าย VC ในลอนดอน ในภาษาแม่ของตนได้อย่างง่ายดายเหมือนกับผู้ก่อตั้งในซานฟรานซิสโก เป็นการขจัดอุปสรรคของการระดมทุนข้ามพรมแดนโดยสิ้นเชิง

![ภาษาเกาหลี ดูการสาธิต](demo_ko.png)

### 3. การจัดอันดับ "คะแนนความเกี่ยวข้อง" ที่ชาญฉลาด
โปรแกรมบางโปรแกรมไม่ได้ถูกสร้างขึ้นเท่ากัน อัลกอริธึม `fit_score` ที่กำหนดเองของเราจะประเมินโอกาสและฟองโปรแกรมระดับสูงสุดและใช้งานมากที่สุดไปด้านบนโดยอัตโนมัติ ดังนั้นคุณไม่ต้องเสียเวลาเลื่อนดูลิงก์ที่เสีย

### 4. การกรองและการค้นหาอันทรงพลัง
ต้องการทุน FinTech ใน LatAm หรือไม่? หรือตัวเร่ง AI ในเอเชีย? ใช้ UI ที่ใช้งานง่ายเพื่อกรองตามประเทศ/ภูมิภาค หมวดหมู่ อุตสาหกรรม และกำหนดเวลา

### 5. พอร์ทัล "สมัคร" โดยตรง
เมื่อคุณพบคู่ที่ใช่ คลิก "สมัคร" เพื่อ *โดยตรง* ไปยังพอร์ทัลแอปพลิเคชันอย่างเป็นทางการ

## 💻 สแต็คทางเทคนิคและสถาปัตยกรรม

-**แบ็กเอนด์:**Python (Flask, SQLAlchemy)
-**ฐานข้อมูล:**SQLite (อัปเดตเป็นกลุ่มในธุรกรรมเดี่ยวที่ปรับขนาดได้มากเกินไป)
-**ส่วนหน้า:**HTML5, CSS3 (Custom Vanilla CSS, Glassmorphism UI), Vanilla JavaScript
-**การแปล:**Flask-Babel และ `deep-translator` (Google Translate API) สำหรับการแปลแบบไม่พร้อมกันแบบเรียลไทม์
-**กลไกข้อมูล:**โปรแกรมรวบรวมข้อมูล Python แบบอะซิงโครนัส (`aiohttp`, `asyncio`) ใช้การสร้างขั้นตอนสำหรับการฉีดข้อมูลขนาดไฮเปอร์สเกลขนาดใหญ่

## 🛠️ วิธีเรียกใช้ในพื้นที่

1.**โคลนพื้นที่เก็บข้อมูล:**
   ``` ทุบตี
   โคลนคอมไพล์ https://github.com/genoxdeveloper/Awesome-Startup-Global-Explorer.git
   cd Awesome-Startup-Global-Explorer
   ```

2.**ติดตั้งการพึ่งพา:**
   ``` ทุบตี
   pip ติดตั้ง -r ข้อกำหนด.txt
   ```

3.**เริ่มต้นและเรียกใช้:**
   ``` ทุบตี
   หลาม app.py
   ```
   *แอปจะเริ่มต้นฐานข้อมูลโดยอัตโนมัติ เริ่มการสร้างข้อมูลพื้นหลัง (สร้างบันทึกมากกว่า 35,000 รายการ) และโฮสต์เซิร์ฟเวอร์ท้องถิ่นบน `http://localhost:5000`.*
