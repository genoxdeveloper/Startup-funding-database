![Demo Dashboard](demo_en.png)

##  为什么创建这个项目？

作为一家位于韩国首尔的初创企业（[Genox Holdings](https://genoxholdings.com)），我们曾花费大量时间在几十个国家的政府门户网站、风险投资数据库和加速器网站上寻找机会。我们最初是为内部使用而构建了这个工具，现在我们将其开源，因为我们坚信：**每一位创始人都有权获得全球范围内的机会**，而不仅仅是硅谷的创业者。

##  这个网站能为您做什么？

跨境寻找合适的资金或支持项目可能会让人不知所措。此应用程序通过为您分担繁重的工作来解决这个问题：

### 1.  **即时发现全球资金**
探索一个庞大且持续更新的数据库，包含：
- **政府补助**（例如美国的 SBIR，英国的 Innovate UK，韩国的 K-Startup，欧洲的 Horizon Europe）
- **风险投资(VC)和加速器**（Y Combinator，Techstars，500 Global 以及数千个区域性微型 VC）
- **企业开放式创新(OI)** 项目
- **云服务积分和福利**（AWS Activate，Google for Startups）
- **人才引进和增长** 举措（创业签证、科技园区入驻）

### 2.  **智能“相关性评分”排名**
并非所有项目都同样有价值。我们定制的 `fit_score` 算法会评估所有机会，并自动将最顶级、最活跃的项目排在前面，以免您将时间浪费在无效链接上。

### 3.  **原生多语言支持**
创业生态系统不应被语言障碍所限制。只需在顶部导航栏点击一下，即可无缝地将整个平台和 33,000 多个项目描述翻译为各种语言。

### 4.  **强大的过滤和搜索**
需要拉美地区的金融科技(FinTech)资助？还是亚洲的 AI 加速器？使用直观的用户界面按以下条件进行筛选：
- **国家 / 地区**
- **类别**（政府补助、VC、企业等）
- **行业**（人工智能、金融科技、SaaS、深度科技、健康科技、Web3、清洁技术等）
- **截止日期**

### 5.  **直达“申请”门户**
当您找到完美匹配的项目时，点击“Apply(申请)”即可直接跳转至官方申请门户或国家拨款数据库。
