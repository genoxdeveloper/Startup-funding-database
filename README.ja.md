**Awesome Startup Global Explorer**へようこそ。これは、グローバルなスタートアップ エコシステムをナビゲートするための究極のゲートウェイです。シード資金を求めている初期段階の創業者であっても、政府の補助金やトップレベルの VC を求めて規模を拡大しているテクノロジー企業であっても、このプラットフォームは**188 か国以上にわたる 35,000 以上の資金調達の機会**を一元管理します。

![デモダッシュボード](demo_en.png)

## 🏢 Genox & Partners について

**Genox**は、韓国のソウルに拠点を置く革新的なテクノロジー ベンチャーです。私たちは、世界的なスタートアップ企業を支援するデータ駆動型のソリューションとプラットフォームを構築します。私たちは、機会へのアクセスを民主化し、国境を打ち破り、イノベーションを加速すると信じています。

このオープンソース プロジェクトは、世界中の開発者とクリエイターに力を与えることを目的としたプレミア プラットフォームである**[Buygit.com](https://buygit.com)**とのパートナーシップにより誇りを持ってサポートされ、共同開発されています。 Genox と Buygit.com は協力して、次世代のユニコーン スタートアップを構築するために必要なツールとデータを創業者に提供することに取り組んでいます。

私たちは、数十カ国の政府ポータル、VC データベース、アクセラレータ サイトを調査するのに数え切れないほどの時間を費やしました。私たちは最初にこのツールを社内で構築しましたが、**すべての創業者はシリコン バレーだけでなく世界的な機会にアクセスする資格がある**ため、現在はオープンソース化しています。

## 🚀 このサイトで何ができるの?

適切な資金提供やサポート プログラムを見つけるのは、特に国境を越えた場合には困難を伴う場合があります。このアプリケーションは、面倒な作業を代わりに行うことでその問題を解決します。

### 1. グローバル資金を即座に発見
次のような継続的に更新される大規模なデータベースを探索します。
-**政府補助金**(例: 米国の SBIR、Innovate UK、韓国の K-Startup、Horizon Europe)
-**VC とアクセラレーター**(Y Combinator、Techstars、500 Global、および数千の地域のマイクロ VC)
-**コーポレート オープン イノベーション (OI)**プログラム
-**クラウド クレジットと特典**(AWS Activate、Google for Startups)
-**移転と成長**の取り組み (スタートアップビザ、テックハブ滞在)

### 2. ネイティブ多言語サポート (国境を越える)
私たちは、次の大きなユニコーンがどこからでも生まれる可能性があることを認識しています。しかし、世界的なスタートアップのエコシステムは、歴史的には厳しい言語の壁によって守られてきました。重要な資金調達ポータル、複雑な補助金コンプライアンスの詳細、複雑な VC 投資論文は、多くの場合、完全に英語またはローカライズされた方言で埋もれています。これにより、地理と母国語のみに基づいて不当な優位性が生まれます。

創業者を絶対に取り残さないように、当社のプラットフォームは信じられないほど強力で業界をリードする**ネイティブ多言語サポート システム**を備えています。これは単なる UI 変換ではなく、包括的なディープデータ ローカリゼーション パイプラインです。

上部のナビゲーション バーを 1 回クリックするだけで、プラットフォーム全体をシームレスに翻訳できます。これは、**35,000 以上の深くネストされたプログラムの説明、評価基準、業界の焦点、および動的なデータベース エントリ**がすべて即座に次のようにローカライズされることを意味します。

**なぜこれが革命的なのでしょうか?**
ほとんどのスタートアップ データベースは、静的な UI (ボタン、メニュー) のみを翻訳し、実際のデータ (最も重要なデータ) は元の言語のままにします。私たちのプラットフォームは、**デュアルエンジン翻訳アーキテクチャ**を使用してこれを解決します。
1.**静的 UI ローカリゼーション:**「Flask-Babel」を利用し、事前にコンパイルされた「.po」および「.mo」メッセージ カタログを使用して、すべての構造要素、ナビゲーション、およびコア UX がゼロ遅延で完全にローカライズされることを保証します。
2.**動的データ変換:**「ディープトランスレータ」を利用した非同期パイプラインによって強化されます。言語を切り替えると、当社のシステムは、SQLite データベースからライブの非構造化プログラムの説明を取得し、正確なデータ スキーマ、マークダウン形式、および重要な資金調達基準を維持しながら、オンザフライで翻訳します。

これは、ボゴタ、リヤド、またはソウルの創業者が、サンフランシスコの創業者と同じように、米国のSBIR補助金の複雑なコンプライアンス要件や、ロンドンを拠点とするVCネットワークの投資論文を母国語で難なく読むことができることを意味します。これにより、国境を越えた資金調達の摩擦が完全に排除されます。

![韓国語表示デモ](demo_ko.png)

### 3. スマートな「関連性スコア」ランキング
すべてのプログラムが同じように作成されているわけではありません。当社のカスタム `fit_score` アルゴリズムは機会を評価し、最上位層で最もアクティブなプログラムを自動的に最上位に表示するため、デッドリンクをスクロールする時間を無駄にする必要はありません。

### 4. 強力なフィルタリングと検索
中南米での FinTech 補助金が必要ですか?それともアジアのAIアクセラレーターでしょうか？直感的な UI を使用して、国/地域、カテゴリ、業種、期限でフィルタリングします。

### 5. 直接「適用」ポータル
完全に一致するものが見つかったら、「適用」をクリックすると、公式アプリケーション ポータルに *直接* 移動します。

## 💻 技術スタックとアーキテクチャ

-**バックエンド:**Python (Flask、SQLAlchemy)
-**データベース:**SQLite (ハイパースケーラブルな単一トランザクション一括更新)
-**フロントエンド:**HTML5、CSS3 (カスタムバニラ CSS、Glassmorphism UI)、バニラ JavaScript
-**翻訳:**Flask-Babel および `deep-translator` (Google Translate API) によるリアルタイム非同期翻訳
-**データ エンジン:**大規模なハイパースケール データ インジェクションのための手続き型生成を利用する非同期 Python クローラー (`aiohttp`、`asyncio`)。
